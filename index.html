SOBRE MIM

Sou estudante de Análise e Desenvolvimento de Sistemas, com uma paixão em construir soluções inovadoras e eficientes. Minha jornada na programação me levou a explorar o universo Full Stack, onde me sinto à vontade para criar desde a infraestrutura do backend até as interfaces mais interativas no frontend.

Minha expertise abrange o desenvolvimento de aplicações robustas com Java Spring e Django, garantindo sistemas escaláveis e seguros. No lado do cliente, utilizo React e Next.js para construir experiências de usuário ricas e dinâmicas, com estilização moderna através do Tailwind CSS.

Sou proficiente em bancos de dados relacionais como MySQL e PostgreSQL, e tenho forte domínio de ferramentas de controle de versão como Git, assegurando um fluxo de trabalho colaborativo e organizado. Estou sempre buscando aprender e aplicar novas tecnologias para resolver desafios complexos.
